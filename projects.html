Slack Slam

A Slack Integration providing an easy-access key-value store.

A Ruby on Rails app that uses Webhooks to provide a key-value store for users. The users are authenticated using OAuth, and they can save values (such as phrases or gifs) to a corresponding keyword.

CarPal

An Android app for finding the perfect carpool.

Developed using Java with the Eclipse IDE, the application also makes use of an external database (parse.com) to store all of the user information.


Future plans for the app includes the integration of the Google Maps API for displaying pick-up/drop-off locations and the PayPal API for easy payment.


The application was presented by me to the Metrolinx innovations team for consideration as Metrolinx's next carpool app.
